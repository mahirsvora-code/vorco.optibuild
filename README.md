# OptiBuild Feasibility Desk

OptiBuild is a static BOQ + feasibility calculator interface styled to match the VORCO website visual language.

## Run

Open `index.html` in a browser, or run a local static server:

`python3 -m http.server 4180`

Then visit `http://127.0.0.1:4180/`.

## Notes

- Calculator logic is in `app.js`.
- UI styles are in `styles.css`.
- No build step is required.
